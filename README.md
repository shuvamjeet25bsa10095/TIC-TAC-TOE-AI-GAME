# TIC-TAC-TOE-AI-GAME
# Tic-Tac-Toe with AI Opponent

## Overview
Console Tic-Tac-Toe where you play against Minimax AI. No installs needed.

## Features
- Interactive board
- Human input validation
- Optimal AI moves

## Technologies/Tools Used
- Python 3 stdlib only

## Steps to Install & Run
1. git clone https://github.com/[yourusername]/tic-tac-toe-ai.git
2. python game.py

## Instructions for Testing
- python tests.py
- Play: Enter "0 0" for top-left, etc.
